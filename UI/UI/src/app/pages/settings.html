📐 Pipeline Rules
CHURN WINDOW
If a customer goes this many days without placing an order, we treat them as churned (lost).
days
RECENT LOGIN WINDOW
How many days of recent logins count as active engagement. The model treats fewer recent logins as an early churn-warning signal — separate from order activity. (Needs uploaded Login Events to have effect.)
days
REPEAT CUSTOMER THRESHOLD
How many completed orders a customer needs before we count them as a repeat customer.
or more orders
RECENT ORDER GAP WINDOW
How many of a customer's most recent orders we look at to learn their normal buying pace — so we can spot when someone suddenly starts ordering less often.
last orders
